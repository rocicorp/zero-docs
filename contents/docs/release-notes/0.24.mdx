## Install

```bash
npm install @rocicorp/zero@0.24
```

## Upgrading

## Features

- Dramatically faster `whereExists()` with `flip:true` ([doc](/docs/reading-data#join-flipping)).
- Support for `op-sqlite` in React Native ([doc](/docs/react-native#op-sqlite)).
- Preview support for [custom mutators](/docs/custom-mutators#custom-mutate-url) and [synced queries](/docs/synced-queries#custom-get-queries-url).
- Return server errors during synced queries to client ([doc](/docs/synced-queries#handling-errors)).
- Added `analyze()` to the inspector and various other improvements ([doc](/docs/debug/inspector#analyzing-queries)).
- Improved server ZQL APIs, added `node-postgres` and Drizzle support ([doc](/docs/zql-on-the-server)).

## Fixes

- Correctly map primary keys to server names in some edge cases ([PR](https://github.com/rocicorp/mono/pull/4857)).
- Reduce some OTEL error logging to `warn` ([PR](https://github.com/rocicorp/mono/pull/4867)).
- Fix a case of deadlock related to CVR cleanup ([PR](https://github.com/rocicorp/mono/pull/4877)).
- Fix hang on shutdown under Bun CLI (thanks [David](https://github.com/dpeek)!) ([PR](https://github.com/rocicorp/mono/pull/4854)).
- Fix "View already exists" error ([PR](https://github.com/rocicorp/mono/pull/4888), [Discord](https://discord.com/channels/830183651022471199/1415395645203812424/1415395649205305434)).
- Fix rows sometimes missing from `startAt` queries ([PR](https://github.com/rocicorp/mono/pull/4926)]).
- Improve performance of CVR cleanup ([PR](https://github.com/rocicorp/mono/pull/4930)).
- Disable statement-level timeouts that were preventing CVR cleanup from succeeding ([PR](https://github.com/rocicorp/mono/pull/4933)).
- Fix WAL growth when no connected clients ([PR](https://github.com/rocicorp/mono/pull/4954)).
- Fix exception when using local-only queries with `enableLegacyQueries:false` ([PR](https://github.com/rocicorp/mono/pull/4967)).
- Fix inability to use `--flags` with `zero-cache-dev` (thanks [Alizain](https://github.com/alizain)!) ([PR](https://github.com/rocicorp/mono/pull/4966)).
- Reduce ping timeouts during slow queries with better time-slicing ([PR](https://github.com/rocicorp/mono/pull/4989)).

## Breaking Changes

- The `push.url` client parameter was removed in favor of `mutateURL`. A matching `ZERO_MUTATE_URL` config param was also added.
- The `ZERO_ADMIN_PASSWORD` config param is now required in production ([doc](/docs/zero-cache-config#admin-password)).
