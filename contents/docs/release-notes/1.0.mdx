## Installation

```bash
npm install @rocicorp/zero@1.0
```

## Overview

After nearly two years of development, 26 releases, thousands of commits, and hundreds of bugfixes, we are declaring Zero stable.

This means we fully support the Zero API and are committed to maintaining it for the foreseeable future.

Functionally, this release is actually a very minor update to 0.26.2.

## Features

- [**Schema Change Hooks:**](/docs/connecting-to-postgres#custom-publications) Added support for detecting publication changes on Supabase via `COMMENT ON PUBLICATION` statements, working around Supabase's lack of event trigger support for publications.
